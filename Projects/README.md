# Projects

## WiC2
An educational command-and-control system for ESP32 devices featuring Wi-Fi access point with captive portal, admin dashboard, device logging, and Windows agent for system monitoring.
[https://github.com/ik0z/WiC2](https://github.com/ik0z/WiC2)

## kAzlogger-server
A C# application that monitors keystrokes and displays them in real-time via a web interface with authentication and HTML dashboard.
[https://github.com/ik0z/kAzlogger-server](https://github.com/ik0z/kAzlogger-server)

## PEDefeat
An all-in-one detection surface analyzer for PE binaries and scripts. Pinpoints exactly what AV/EDR flags with severity, explanation, and fix suggestions.
[https://github.com/ik0z/PEDefeat](https://github.com/ik0z/PEDefeat)

## kSYSLoki
A secure logging server designed to receive and store logs over TLS connection. Facilitates analysis using YARA rules and supports remote scans with encrypted log transfer.
[https://github.com/ik0z/kSYSLoki](https://github.com/ik0z/kSYSLoki)

## Multi-Sender
A Windows Forms application for sending emails to multiple recipients. Supports importing from text, CSV, and Excel files with attachment support.
[https://github.com/ik0z/Multi-Sender](https://github.com/ik0z/Multi-Sender)

## PDF-Merger
A Windows Forms application to combine multiple PDF documents into a single file. Features adding, removing, and reordering PDFs before merging.
[https://github.com/ik0z/PDF-Merger](https://github.com/ik0z/PDF-Merger)

## proxfinder
Proxy finder utility tool.
[https://github.com/ik0z/proxfinder](https://github.com/ik0z/proxfinder)
